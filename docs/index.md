# AltText.ai Plugin - User Guide

This plugin automatically generates alt text descriptions for your images using AI.

## What It Does

When you upload an image to OctoberCMS:
1. The image is sent to AltText.ai for analysis
2. AI generates a description (usually takes 5-30 seconds)
3. The description is automatically saved to the image

## Quick Start

1. Install the plugin
2. Go to **Settings** â†’ **AltText.ai Settings**
3. Enter your API key from [alttext.ai/account/api_keys](https://alttext.ai/account/api_keys)
4. Save

That's it! Alt text will now be generated automatically when you upload images.
